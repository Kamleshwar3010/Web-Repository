# Modern Weather App

A responsive, feature-rich weather application built with HTML, CSS, and JavaScript. This project provides real-time weather updates, air quality information, and interactive maps.

![Weather App Preview](https://via.placeholder.com/800x400?text=Weather+App+Preview)

## Features

- **Real-time Weather Data**: Fetches current weather conditions including temperature, humidity, wind speed, and cloud cover.
- **Location Search**: Search for weather in any city worldwide.
- **Interactive Map**: Visualizes the location using Mapbox.
- **Air Quality Index (AQI)**: Displays air quality levels with health recommendations.
- **UV Index**: Shows UV radiation levels.
- **Unit Conversion**: Toggle between Celsius/Fahrenheit and kph/mph.
- **Dark/Light Mode**: Automatically detects system preference and offers a manual toggle.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Flexbox, Grid, and CSS Variables for theming.
- **JavaScript (ES6+)**: Asynchronous API fetching and DOM manipulation.
- **WeatherAPI**: Source for weather data.
- **Mapbox GL JS**: Interactive maps.
- **Google Fonts**: Typography (Inter).

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Kamleshwar3010/Web-Repository.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Web-Repository/weather
   ```
3. Open `weather.html` in your browser.
